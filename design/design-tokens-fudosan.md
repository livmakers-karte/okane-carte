# お金のカルテ /fudosan/ 確定デザイントークン（案C）

正典 / design-director（本セッション）確定 / 2026-07-07
相棒: `design/design-brief.md`（面積比・やらないこと）。母屋=案A（緑）・jikabuka=案B（紺）と**5軸で必ず別ブランド**に見せる第3系統。
コンセプト1行: **「土地と建物と街を、一枚のカルテに。等高線の紙面に深いティール、印章は陶土ひとつ ── 測量士の静けさ」**

面積比の絶対指針（design-brief 継承）: 明るい地色 70〜80% / 構造色（スレートティール）15〜25% / 陶土＋金は差し色 5%未満（印章・カルテ指数の針・1pxライン・CTAのみ）。**差し色を面で塗ったら不合格。**

---

## 母屋A・jikabukaB との差別化 5軸

| 軸 | 案A（母屋ポータル） | 案B（jikabuka診断） | **案C（fudosan診断）** |
|---|---|---|---|
| 色相 | 深緑（暖・富の緑） | 深紺（寒・格式） | **スレートティール（青緑・測量/地の色）** |
| 地の温度 | アイボリー（黄み） | パール（青白み） | **石灰ペーパー（灰みの暖白・図面の紙）** |
| 差し色の質感 | シャンパン金 | アンティーク金 | **陶土オレンジ（土・素焼き）＋金は印章1点** |
| 罫/モチーフ | 細罫＋丸い明朝 | 直線罫＋字間広め | **等高線・地形図ライン＋方眼（図面グリッド）** |
| 写真テーマ | 庭園・自然光 | 邸宅内装・建築 | **住宅・街並み・鍵・図面・光の入る部屋** |

差別化ゲート判定: **PASS**。同業「不動産一括査定サイト（原色・煽り・電話誘導）」の畑を一切参照せず、測量図・地形図・建築図面という“設計/測量”の視覚言語を借用。色相をティールに分離、モチーフを等高線に固有化、差し色を陶土に。模倣でなく構造の再構築。

## シグネチャー（1点集中）＝「不動産カルテの発行演出」

- 診断結果で**1枚の“物件カルテ（診断書）”**が発行される。カルテは横罫・右上に発行番号（匿名連番でない＝ハッシュ風の座標ID）・左に等高線の透かし。
- **3レイヤー（積算・収益・税務）が上から順に1本ずつ罫線とともに描き足され**、最後に中央へ「不動産カルテ指数」の針（陶土色）が振れて止まる（reduced-motion では即時表示）。
- 演出は**このカルテ発行の1箇所のみ**に集中。他セクションは静かに保つ（draw-on/回転の乱用禁止）。

## モチーフ実装ルール（等高線・方眼）

- **背景の方眼**: body に `--bg-grid` の 1px グリッド（例 28px 方眼）を極薄で敷く（母屋の横罫線に相当する“地”の演出。図面の方眼紙）。彩度・濃度は地色に沈める。
- **等高線**: ヒーロー右上・カルテ左・区切りに、SVG/CSSの同心の不定曲線（`--teal-3` の 1px、opacity .12〜.2）を透かしとして配置。地形図の等高線＝“土地”の主題を象徴。写真ゼロでも成立する装飾の主役はこの線画。
- **区切り .orn**: 中央に測量点（◎＝基準点マーク）＋左右にティールの細線。金でなくティール/陶土。

## 金・差し色の総量チェック（compliance-qa 自己点検）

- [ ] 陶土（clay）の背景塗り（面）が無い。陶土は 印章 / カルテ指数の針 / 1pxライン / sel状態の淡地 / CTA に限定。
- [ ] 金（gold）は emblem/認定印の SVG 内のみ。ボタン地・見出しに金の面を使わない。
- [ ] 1画面あたり（陶土＋金）の可視面積が 5%未満。
- [ ] 濃色ブロックは「カルテ指数の重心」と「税務ギャップ図の1帯」に限る（`--teal-ink`）。地色が石灰ペーパーで70%以上。
- [ ] 写真は不透明度 0.05〜0.12 の透かし・彩度抑制。等高線/方眼の線画が装飾の主役。
- [ ] A（緑）/B（紺）/C（ティール）を並べて別ブランドに見える（色相・地温度・差し色・モチーフ・写真の5軸）。

## レスポンシブ / a11y（design-brief 継承）

- ブレークポイント: 760px（1カラム化）/ 640px（透かし縮小）/ 520px（grid→1列）/ 最小検証 **360px**（全見出し clamp 流動・はみ出し根絶）。
- フォーカス可視: `:focus-visible{outline:2px solid var(--teal); outline-offset:2px}` を全 interactive に。
- `@media (prefers-reduced-motion: reduce)`: カルテ発行アニメ・針の振れ・パララックスを無効化し即時最終状態。
- コントラスト: 本文 `--ink` on `--paper/bg` ≥ 4.5:1。陶土 `--clay` on 白は大きな文字/罫のみ（本文には使わない）。
